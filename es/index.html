Fantásticas funciones

Una aplicación que logrará quedarse contigo para siempre.

Añade con solo escanear el código de barras

Basta con abrir MyShoppingList y escanear el código de barras para que se añada a tu lista de la compra.

Una cuenta, todo tu hogar

Puedes utilizar una sola cuenta para todo tu hogar, de esta manera cualquier miembro podrá añadir a la misma lista el producto deseado.

Siempre conectado

No importa desde que dispositivo accedas, siempre podrás continuar tu lista estés donde estés.

Al día en productos

En MyShoppingList siempre contarás con los últimos productos disponibles y actualizados.

Busca el producto que desees añadir

Siempre que quieras podrás buscar el producto que quieras añadir, aunque no tengas el código de barras.

Compatible con todos los dispositivos

MyShoppingList es una aplicación multiplataforma, por lo cual podrás utilizarla en Android e iOS.
